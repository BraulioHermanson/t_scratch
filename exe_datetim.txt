Exercicio 1-
Oferecendo desconto para cliente com base na ultima compra
- Suponha que voce esta trabalhando para uma empresa que deseja rastrar a atividade do cliente. 
Uma metrica que eles estão interessados é o tempo que passou desde a ultima transação do cliente.
Se for muito tempo, eles podem oferecer um desconto para o cliente.
Crie um script Python que mostra quanto tempo se passou desde a ultima compra do cliente(10 de maio de 2023).
Se faz mais de 30 dias, mostre uma mensagem oferecendo um desconto para o cliente
